# QR-Based Attendance System

A secure attendance management system built with an Android client and web backend.  
It uses QR scanning, GPS validation, and selfie verification to reduce proxy attendance.

## Features
- QR-based attendance marking
- Time-limited QR validation
- GPS location verification
- Selfie capture and validation
- Lecturer dashboard
- CSV export of attendance records

## Tech Stack
- Android: Kotlin, CameraX, ML Kit
- Backend: Node.js, Express
- Frontend: HTML, JavaScript
- Data handling: CSV / local storage / database

## Modules
### Android App
- Student login / roll number input
- GPS permission and validation
- Selfie verification
- QR scanning

### Web Dashboard
- Session creation
- QR generation
- Attendance records view
- CSV export

## Project Structure
- `android-app/` → Android mobile application
- `web-version/` → Web dashboard and backend
